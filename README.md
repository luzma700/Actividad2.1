# Actividad2. - Introducción a JavaScript en el desarrollo backend

# Objetivo
Practicar cuatro formas básicas de mostrar un mensaje en JavaScript dentro de un contexto web:
1. Alerta emergente (`alert`)
2. Inserción en el documento (DOM)
3. Impresión en consola (`console.log`)
4. Mensaje dentro de un `<div>`

Este ejercicio sirve como base para integrar JavaScript con Node.js/Express en el desarrollo backend.
